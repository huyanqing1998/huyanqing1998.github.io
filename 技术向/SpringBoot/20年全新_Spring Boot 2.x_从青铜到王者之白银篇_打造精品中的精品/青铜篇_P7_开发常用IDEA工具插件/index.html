    安装插件

File -> Settings… -> Plugins -> Marketplace

搜索，安装，重启IDEA。

Codota

  1. 代码自动补全

  1. 参考案例

Auto filling Java call arguments

new新对象时自动补齐构造器参数。

GsonFormat

根据json自动生成对应的JavaBean类。

使用

创建一个空类，Alt+ S 打开窗口，粘入json数据。

Rainbow Brackets

彩虹括号，更易于区分代码块关系。

Maven Helper

分析Jar包关系，解决依赖冲突。

作用于 pom.xml 文件。
