认识 XPath

1. 什么是 XPath

  1. 解析 XML 的一种语言（HTML 其实是 XML 的子类），广泛用于解析 HTML 数据；
  2. 几乎所有语言都能使用 XPath，比如 Java 和 C 语言；
  3. 除了 XPath 还有其他手段用于 XML 解析，比如：BeautifulSoup、lxml、DOM、SAX、JSDOM、DOM4J、minixml 等。

2. XPath 语法

XPath 语法 3 大类：

  • 层级：/ 直接子集、// 跳级；
  • 属性： @ 属性访问；
  • 函数：contains()、text()。
