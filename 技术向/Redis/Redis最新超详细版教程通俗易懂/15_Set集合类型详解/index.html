15 Set集合类型详解

Redis最新超详细版教程通俗易懂

Set（集合）

set中的值是不能重复的。

SADD 向集合里添加一个值

SMEMBERS 查看集合里面的值

SISMEMBER 集合内是否存在某个值

SCARD 获取集合内容元素的个数

SREM 移除集合里的某个元素

SRANDMEMBER 随机从集合里抽取出一个元素

有可能抽出重复的。

SPOP 随机弹出一个元素，并在集合里将其删除。

SMOVE 将集合里的元素移动至另一个集合中

参数：集合1 集合2 需要移动的元素

数学集合：

  • 差集
  • 交集
  • 并集
    SDIFF 差集

集合1 - 集合2

SINTER 交集

两个集合共同的

共同好友可以

SUNION 并集

集合1 + 集合2
