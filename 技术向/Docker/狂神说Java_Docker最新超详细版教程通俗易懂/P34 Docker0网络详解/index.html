P34 Docker0网络详解

【狂神说Java】Docker最新超详细版教程通俗易懂

原理

  1. 我们只要安装了docker，就会有一个网卡docker0。

    lo：回环，eth0：本机网卡，docker0：docker网卡

  2. 我们每启动一个docker容器，docker就会给docker容器分配一个ip。网卡桥接模式，使用veth-pair技术。

    什么是veth-pair

    就是一对对虚拟设备接口，他们都是成对出现。

  3. 测试 tomcat01 和 tomcat02 是否可以ping通

    结论：容器之间是可以互相ping通的。

    所有容器在不指定网络的情况下，都是由docker0路由，docker会给我们的容器分配一个默认可用的IP。

    Docker 中的所有的网络接口都是虚拟的，虚拟的转发效率高。

    只要容器删除，对应的网桥一对就没了。
