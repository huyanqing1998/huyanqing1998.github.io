P36 自定义网络

【狂神说Java】Docker最新超详细版教程通俗易懂

基础

–link 和 自定义网络 都叫做 容器互联。

网络模式

  • bridge：桥接，docker默认

  • none：不配置网络

  • host：和宿主机共享网络

  • container：容器网络连通

测试

自定义网络

使用 docker network ls 可以查看所有的网络信息，如果创建成功，可以看到mynet网络。

使用 docker network inspect mynet 查看mynet网络详细信息。

容器放入自定义网络

适用场景
