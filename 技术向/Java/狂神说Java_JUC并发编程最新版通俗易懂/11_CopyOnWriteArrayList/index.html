ArrayList 是线程不安全的

使List线程安全的三种方式

  • List<String> list = new Vector<>(); 效率低，读写 synchronized 加锁，悲观锁
  • List<String> list = Collections.synchronizedList(new ArrayList<>()); 不安全List 转 安全List
  • List<String> list = new CopyOnWriteArrayList<>(); 写入时复制，仅对写入 Lock 加锁
