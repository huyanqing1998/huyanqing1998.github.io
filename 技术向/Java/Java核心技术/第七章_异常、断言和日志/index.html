第七章 异常、断言和日志

7.1.1 异常分类

术语：非受查异常和受查异常

  • 非受查异常：派生于Error或RuntimeException
  • 受查异常：其他所有的异常称为受查异常

一个方法必须声明所有可能抛出的受查异常，非受查异常要么不可控制（Error），要么就应该避免（RuntimeException）。当方法没有声明所有可能发生的受查异常，编译器会发出一个错误消息。

RuntimeException属于程序本身的问题，应避免出现。
