4.th:utext 和 th:id

th:utext 和 th:id都是thymeleaf的springEL语法。

th:utext：将获取到的字符串解析，把字符串内的html语言加载至页面。相当于js中的html()方法。th:text就是text()方法。

th:id：给元素加入id属性。

紫色部分，通过text()方法先将数据流内的html标签洗掉，然后再通过html()方法加载至页面。当然使用正则表达式来清洗数据库数据更好。
