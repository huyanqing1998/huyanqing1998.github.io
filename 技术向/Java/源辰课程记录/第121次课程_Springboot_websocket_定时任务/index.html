2.定时任务

C76S3PlySpringBootApplication.java

启动器内加入**@EnableScheduling**注解。

MyWebSocket.java

任意一个存在**@Component**注解的类，都可以将定时器方法代码加入其中。

定时器方法注解：**@Scheduled(cron=”/5 * 10-13 * * ?”)*

其中： */5 每五秒执行一次。    10-13当天10:00至13:00内执行。
